========================================
配置文件 (config.json)
========================================

## settings — 渲染环境设置

renderMode     - 渲染模式: solid / edges / overlay
edgeLineWidth  - 边缘粗细
thresholdAngle - 硬边阈值 (度)
solidOpacity   - 实体透明度 (0-1)
wfOpacity      - 线框透明度 (0-1)
bgColor        - 背景颜色 (hex)
ambientIntensity - 环境光强度
keyIntensity   - 主光源强度
fillIntensity  - 补光强度
hemiIntensity  - 半球光强度
showGrid       - 显示网格
showAxes       - 显示坐标轴
showCenterDot  - 显示中心点
showBBox       - 显示包围盒
autoRotate     - 自动旋转
flatShading    - 平面着色
sobel          - Sobel 边缘检测后处理
bloom          - 泛光效果
fxaa           - FXAA 抗锯齿
edgeSeeThrough - 边缘穿透显示
solidSeeThrough - 实体穿透显示
bloomThreshold - 泛光阈值
bloomStrength  - 泛光强度
bloomRadius    - 泛光半径
cameraNear     - 摄像机近裁面
cameraFar      - 摄像机远裁面
cameraType     - 相机类型: perspective / orthographic
showLabels     - 显示模型标签
labelFontSize  - 标签文字大小 (10-50)
labelHeight    - 标签高度偏移 (0.1-10)
viewPreset     - 视图大小: small / medium / large
camPos         - 相机位置 {x, y, z}
camTgt         - 相机目标 {x, y, z}

## models — 模型配置 (key 为模型文件名)

name           - 模型文件名
position       - 位置 {x, y, z}
scale          - 缩放 {x, y, z}
rotation       - 旋转 {h, p, b} (度)
colors         - 颜色状态 { normal, hover, selected }
  每状态: { edge: hex, background: hex }
materialColors - 每材质颜色 {材质名: {normal, hover, selected}}
meshVisibility - mesh 可见性 {mesh名: true/false}
label          - 标签文字
labelVisible   - 标签可见
labelPerHeight - 标签每高度
labelFontSize  - 标签字体大小

## sceneCameras — 场景摄像机

id             - 唯一标识
name           - 摄像机名称
position       - 位置 {x, y, z}
rotation       - 旋转 {h, p, b} (度)
fov            - 视场角
near / far     - 裁面
isOrtho        - 正交/透视
zoom           - 正交缩放
colors         - 摄像机颜色
